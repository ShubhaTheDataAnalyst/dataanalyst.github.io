# Superstore Sales Dashboard

## Project Overview
This project analyzes the Superstore dataset and creates an interactive sales dashboard to track business performance. The dashboard helps identify sales trends, profit performance, and top-performing regions.

## Dataset
The dataset used in this project is the Superstore dataset which contains information about:
- Orders
- Sales
- Profit
- Customer segments
- Product categories
- Regions

## Tools Used
- Power BI
- Excel
-SQL
- Data Cleaning
- Data Visualization

## Dashboard Features
The dashboard includes the following visualizations:
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top 10 Products by Sales

## Key Insights
- The West region generated the highest sales.
- Technology category produced the highest profit.
- Sales increased significantly during the holiday season.
- Some products have high sales but low profit margins.
